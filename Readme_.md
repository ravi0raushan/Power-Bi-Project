
#  Power BI Sales Analysis Dashboard

## Project Overview

This project focuses on analyzing sales performance using Power BI by transforming raw data into meaningful business insights. The dashboard provides a comprehensive view of revenue trends, customer behavior, and product performance, enabling data-driven decision-making.

---

## Problem Statement

Businesses often face challenges in:

* Tracking overall sales performance
* Identifying high-performing products and regions
* Understanding customer purchasing behavior
* Detecting profit and loss patterns

This project solves these challenges by building an interactive dashboard that delivers actionable insights.

---

## Dataset Used

* **Orders.csv** → Contains order-level data (sales, customer, region, etc.)
* **Details.csv** → Contains product/category-level information

---

## Tech Stack

* Power BI
* Power Query (Data Cleaning & Transformation)
* Data Modeling
* DAX (Data Analysis Expressions)

---

## Features

* Sales & Profit Overview Dashboard
* Time-based analysis (monthly/yearly trends)
* Region-wise performance analysis
* Category & Sub-category insights
* Customer behavior analysis
* Profit vs Loss visualization

---

## Key Insights

* Identified top-performing categories contributing maximum revenue
* Found regions with high sales but low profit margins
* Discovered seasonal trends affecting sales performance
* Highlighted loss-making segments
* Analyzed customer purchasing patterns

---

## Business Impact

* Helps optimize product and sales strategy
* Improves decision-making with visual insights
* Identifies areas of profit leakage
* Enables targeted marketing strategies
